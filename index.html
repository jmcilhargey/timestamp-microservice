Timestamp Microservice

By Joe McIlhargey | GitHub

https://timestamp-service-api.herokuapp.com/
  • Enter a date in either standard format (January 1, 2016) or unix format (1451606400)
  • Date can be entered into form field or website URL
  • The service will return a JSON with the date in standard and unix format
  • If date is invalid format, the service will return JSON with null values

Example Input

Einstein's B-day / PI Day

https://timestamp-service-api.herokuapp.com/March, 14 2016
https://timestamp-service-api.herokuapp.com/1457913600

Example Output

{ "unix": 1457913600, "standard": "March 16, 2016" }
